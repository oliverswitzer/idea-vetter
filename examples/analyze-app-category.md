# Example: Analyze an App Category

## Prompt

```
Analyze the habit tracking app market. I want to understand: who are the top
players, what do users love and hate, what's the pricing landscape, and where
are the gaps? Focus on iOS but include major Android-only players if relevant.
```

## What to Expect

The system will:

1. **Search** app stores for "habit tracker", "habit builder", "daily routine", "streak tracker"
2. **Profile** top apps: Habitica, Streaks, Habitify, Productive, Loop Habit Tracker, etc.
3. **Mine reviews** for recurring complaints (too complex, missing features, subscription fatigue, poor widget support)
4. **Map pricing**: free tier availability, subscription ranges, lifetime deals
5. **Identify gaps**: e.g., habit tracking for teams, habit tracking for specific niches (fitness, sobriety, ADHD), simpler alternatives

## Follow-up Prompts

```
Which of these gaps has the least competition and the strongest user demand signal?
```

```
Pull Google Trends data for "habit tracker app" vs "routine planner app"
vs "streak tracker app" over the past 2 years.
```

```
Are there any habit tracking apps that specifically target people with ADHD?
How are they positioned and reviewed?
```
