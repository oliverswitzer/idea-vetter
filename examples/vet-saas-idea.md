# Example: Vet a SaaS Idea

## Prompt

```
Vet this idea: A SaaS tool that helps small e-commerce brands (Shopify stores
doing $10K-$500K/month) automate their product return process. Right now most
of them handle returns via email, which is slow and error-prone. The tool would
provide a branded returns portal, automated label generation, and analytics on
return reasons.
```

## What to Expect

The system will:

1. **Frame**: Restate the idea, identify the target user (small Shopify merchants), job-to-be-done (process returns efficiently), and list assumptions
2. **Research**: Search for existing return management tools (Loop, Returnly, AfterShip Returns), check Shopify App Store listings, look for merchant complaints in r/shopify and Shopify Community forums, pull Google Trends data for "shopify returns app"
3. **Synthesize**: Cluster pain points (email chaos, no analytics, manual label creation), score them, map workarounds
4. **Challenge**: Note that Loop and Returnly are well-funded, question whether small merchants will pay $50+/month, flag that Shopify's native returns feature is improving
5. **Recommend**: Suggest a wedge (e.g., focus on return reason analytics for brands with high return rates) or recommend niching down
