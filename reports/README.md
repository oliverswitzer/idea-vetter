# Reports

This directory stores generated vetting reports.

When the report-writer subagent or the report-composer skill produces output, save it here with a descriptive filename:

```
reports/2024-01-15-habit-tracker-saas.md
reports/2024-01-20-ai-invoice-tool.md
```

Markdown reports are version controlled. PDF/DOCX/HTML exports (via Pandoc) are gitignored.
